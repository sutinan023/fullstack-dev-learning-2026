# ui.md

คุณคือ Senior Frontend Developer ผู้เชี่ยวชาญ Next.js และ UI Design

สร้าง UI เว็บไซต์แนว YouTube Clone สำหรับฟังเพลงและดูวิดีโอ

## Tech Stack

* Next.js
* Supabase Auth
* Supabase Database
* Deploy บน Vercel

## Scope ที่ต้องมีเท่านั้น

* หน้า Home แสดงรายการวิดีโอ/เพลง
* หน้า Watch สำหรับดูวิดีโอ
* หน้า Artist Detail
* หน้า Login ด้วย Supabase Auth
* OAuth Login ด้วย GitHub
* หน้า User Watch History

## UI Requirement

* หน้าตาคล้าย YouTube
* Layout เรียบง่าย ใช้งานง่าย
* Responsive รองรับ Desktop และ Mobile
* มี Navbar ด้านบน
* มี Sidebar หรือ Menu สำหรับนำทาง
* Card แสดงวิดีโอ เช่น thumbnail, title, artist name
* ปุ่ม Login / Logout
* แสดงสถานะผู้ใช้เมื่อ Login แล้ว

## ห้ามทำ

* ห้ามเพิ่มระบบ upload video
* ห้ามเพิ่มระบบ comment
* ห้ามเพิ่มระบบ like/dislike
* ห้ามเพิ่มระบบ subscription
* ห้ามเพิ่มระบบ playlist
* ห้ามเพิ่ม feature อื่นนอก Requirement
